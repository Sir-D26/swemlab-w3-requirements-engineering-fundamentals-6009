Week 3 Lab: Requirements Engineering for Campus Event Finder
Scenario

This lab focuses on defining the requirements for a proposed mobile application called Campus Event Finder. The goal of the application is to help university students easily discover and attend events happening on or around campus, such as club meetings, sports events, academic workshops, and guest lectures. The app will allow students to view event listings, filter events by category, and mark events they are interested in so they can receive reminders.
